# annoy-jorsh
Discord.Net bot to do fun things for friends

# Implementation
`git clone https://github.com/aaronmott/annoy-jorsh.git`
add a appsettings.json file to the project root with the following:
~~~
{
    "DiscordToken":"YourDiscordBotTokenHere"
}
~~~
`dotnet restore`
`dotnet build`
`dotnet run`
